# Container.cr
Simple IoC Container.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     radbas-container:
       github: radbas/container.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "radbas-container"
```
